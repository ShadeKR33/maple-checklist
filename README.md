# 🍄 메이플 주간 체크리스트

메이플스토리 주간 컨텐츠를 요일별로 관리하는 체크리스트 웹앱입니다.  
목요일 보스 초기화 기준으로 설계되어 있으며, 획득 메소와 소요 시간을 실시간으로 추적합니다.

<br>

## 미리보기

![체크리스트 화면](https://[유저네임].github.io/maple-checklist)

<br>

## 주요 기능

- **요일별 할 일 관리** — 목~일(집중) / 월~수(여유) 구분
- **실시간 통계** — 완료 항목, 획득 메소, 남은 시간 즉시 반영
- **진행률 게이지** — 주간 달성률 시각화
- **자동 주차 감지** — 목요일 초기화 기준으로 현재 주차 자동 계산
- **데이터 유지** — localStorage 기반으로 새로고침해도 체크 상태 보존
- **다크모드 지원** — 시스템 설정에 따라 자동 전환
- **모바일 최적화** — 스마트폰에서도 편하게 사용 가능

<br>

## 주간 스케줄

| 요일 | 컨텐츠 | 소요시간 | 구분 |
|------|--------|--------|------|
| 목요일 | 보스돌이 안셰인 + 사냥 | 3시간 | 집중 |
| 금요일 | 보스돌이 엘나영 + 사냥 | 3시간 | 집중 |
| 토요일 | 보스돌이 진호참 + 엥글러 1회차 + 사냥 | 2시간 30분 | 집중 |
| 일요일 | 엥글러 2회차 + 사냥 | 1시간 30분 | 집중 |
| 월요일 | 사냥 | 1시간 | 여유 |
| 화요일 | 사냥 | 1시간 | 여유 |
| 수요일 | 사냥 | 1시간 | 여유 |

**주간 총 수익: 약 42억 5천만 메소 / 총 소요시간: 13시간**

<br>

## 사용 방법

1. 페이지 접속
2. 완료한 컨텐츠의 체크박스 클릭
3. 상단 통계에서 획득 메소 및 진행률 확인
4. 다음 주 초기화 버튼으로 새 주차 시작

<br>

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라, 별도 라이브러리 없음)
- localStorage — 브라우저 데이터 영속성
- GitHub Pages — 무료 정적 호스팅

<br>

## 로컬 실행

별도 설치 없이 `index.html` 파일을 브라우저에서 바로 열면 됩니다.

```bash
git clone https://github.com/[유저네임]/maple-checklist.git
cd maple-checklist
open index.html  # macOS
# 또는 index.html 파일을 브라우저로 드래그 앤 드롭
```
